# Class01 - 用户信息管理平台

简易用户信息管理平台的登录功能，基于 Python Flask 框架。

## 功能

- 用户登录（含默认管理员账号泄露）
- 用户信息展示（含密码明文展示）
- 会话管理（登录/登出）

## 快速启动

```bash
pip install flask
python3 app.py
```

访问 http://localhost:5000
